# Ray_Tracer

Basic ray tracer implemented in C++.
Results are output to an image file.

## Features

- Raycasting
- Shadows

### Shading
Currently supports up to 3 types of shading.
- Flat shading 
- Phong shading
- specular shading.

### Intersections
Capable of calculating
- Sphere intersections
- Plane intersections
